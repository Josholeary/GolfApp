About:

This app allows players to come together and set up a scorecard which they can all join to track their scores across the course. Once set-up a user can input their scores on each hole and see other users on a live leaderboard including what hole they are on and their current score.



How to use:


Host a game

To set-up a game, give your game session a name, enter the number of holes you wish to play and set a password. Other players can then enter the name and password in order to start tracking their scores.

Get started hosting
Join a game

To join a game enter the name that will be linked to your score card, the game session name and the session password and begin inputting your own scores on every hole number as well as seeing scores from around the course.

Join an existing game
