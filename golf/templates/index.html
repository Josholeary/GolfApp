Welcome to Golf Score Tracker


About:

This app allows players to come together and set up a scorecard which they can all join to track their scores across the course. Once set-up a user can input their scores on each hole and see other users on a live leaderboard including what they got on each hole.



How to use:

Host a game

Go to the "Setup a new game" page and enter your desired name, the number of holes the group intends to play and the password you want to set for the game, other players can then search for your name and enter the password in order to start tracking their scores.

Join a game

Go to the "Join a game" page enter the hosts name, your name for the scorecard and the session password and see scores from around the course.
